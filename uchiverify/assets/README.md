# Assets Directory

## Required Images

Place these image files in this directory:

### 1. `logo.png` - UChiVerify Bot Logo
- The UChiVerify phoenix logo with maroon checkmark
- Format: PNG (preferably with transparent background)
- Recommended size: 512x512px (square aspect ratio)
- Used in: Hero section and Discord embed mockup

### 2. `uchicago_logo.png` - University of Chicago Logo
- The official University of Chicago logo
- Format: PNG with transparent background recommended
- Should include the shield and "THE UNIVERSITY OF CHICAGO" text
- Used in: Sign-in page animation
- Can be obtained from: UChicago branding guidelines

## Adding the Images

Simply save or copy your image files to:
```
assets/logo.png
assets/uchicago_logo.png
```

The website is configured to display these images automatically.
